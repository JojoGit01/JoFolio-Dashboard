# 🖥️ Jo Dev OS — Portfolio Dashboard - The One

Un portfolio conçu comme une **véritable application**.  
Au lieu d’un site classique qui scroll, ce portfolio fonctionne comme un **workspace de développeur**.

> L’objectif : être mémorable et montrer de vraies compétences frontend.

---

## ✨ Concept

Ce portfolio est pensé comme un **environnement de travail interactif**.

Il propose une expérience différente pour les recruteurs et développeurs :

- Layout fixe type dashboard
- Vue centrale dynamique
- Micro-interactions partout
- Sensation d’application réelle

Desktop → ressemble à un SaaS  
Mobile → ressemble à une app native

---

## 🎯 Pourquoi ce portfolio existe

La majorité des portfolios sont statiques.

Celui-ci démontre :
- la réflexion UX
- l’architecture frontend
- les animations & micro-interactions
- une approche produit réelle

Ce portfolio est à la fois un **CV** et un **produit**.

---

## 🧠 Fonctionnalités principales

### 🪟 Layout style application
- Sidebar et topbar fixes
- Vue centrale dynamique
- Transitions fluides entre les sections
- Chargement partiel (pas de reload complet)

---

### 🎮 Expérience interactive
- Cards avec effet 3D au hover
- Compteurs animés
- Avatar assistant interactif
- Skeleton loaders
- Transitions fluides entre les vues

---

### 🧩 Explorateur de compétences
Système interactif des skills :

- Catégories (Frontend, Backend, Mobile, DevOps…)
- Clic sur une techno → panneau de détails
- Projets liés à chaque technologie
- Statistiques d’expérience

---

### 🚀 Présentation des projets
- Cards interactives
- Mise en avant de la stack technique
- Liens GitHub & démos live
- Preview animée au hover

---

### 📊 Données développeur en temps réel
Connexion à l’API GitHub :
- Activité de commits
- Technologies utilisées
- Statistiques dynamiques

---

### 🤖 Mode recruteur
Visite guidée automatique du portfolio :
- Mise en avant des sections importantes
- Explication rapide des projets
- Optimisé pour une lecture rapide
